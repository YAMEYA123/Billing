阿里云 Token Plan 成员、坐席与调用链路

外部可观察架构 · 控制面与数据面分离 · 截至 2026-07-20
实线为官方明确行为；黄色虚线为逻辑推导或未公开边界。

Key 与成员

  • • 坐席分配给成员后生成专属 Key
  • • 请求只携带 Key，不显式携带成员 ID
  • • 成员承担管理、归因与审计语义
  • • Key 底层绑定字段未公开

额度与故障域

  • • 先扣成员坐席，再扣企业共享包
  • • 额度耗尽不等于 Key 被撤销
  • • 技术限流仍按主账号聚合
  • • 一个成员突增可能影响其他成员

生命周期边界

  • • 重置 Key、移出组织：旧 Key 立即失效
  • • 回收坐席：成员失去使用权
  • • 回收后的精确吊销时延未公开
  • • 重新分配生成新 Key，不复用旧 Key
